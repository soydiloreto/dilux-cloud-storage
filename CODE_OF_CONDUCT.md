# Code of Conduct

This project adopts the **Contributor Covenant** version 2.1 as its code of conduct.

The full text is available at the official source:

🔗 **https://www.contributor-covenant.org/version/2/1/code_of_conduct/**

A translated version (Spanish) is available at:

🔗 **https://www.contributor-covenant.org/es/version/2/1/code-of-conduct/**

## Scope

The Contributor Covenant applies within all project spaces — the GitHub repository (issues, pull requests, discussions), and any other communication channel maintained by the project. It also applies when an individual is officially representing the project in public spaces.

## Reporting

Instances of behavior that violate the Code of Conduct can be reported privately to the maintainer at **pablodiloreto@gmail.com**. All reports will be reviewed and investigated promptly and fairly. The maintainer is obligated to respect the privacy and security of the reporter.

## Enforcement

The Contributor Covenant 2.1 includes "Enforcement Guidelines" that the maintainer follows when responding to reports. See the linked official text for the full guidelines.
